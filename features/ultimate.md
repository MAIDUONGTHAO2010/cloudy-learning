# 💎 Bản Ultimate

Bản **Ultimate** là giải pháp toàn diện cấp doanh nghiệp — hỗ trợ multi-tenant, white-label, API mở và hệ sinh thái affiliate. Phù hợp cho các tổ chức giáo dục lớn, EdTech startup hoặc đối tác muốn xây dựng nền tảng riêng trên Cloudy Learning.

> ✅ Bao gồm tất cả tính năng của [Normal](./normal.md) và [Pro](./pro.md)

---

## 🏢 Doanh nghiệp & Multi-tenant

| Tính năng | Mô tả |
|-----------|-------|
| Multi-tenant | Nhiều tổ chức / trường học vận hành độc lập trên cùng hệ thống |
| Custom Domain | Mỗi tổ chức dùng domain riêng (vd: `learn.university.edu`) |
| White-label | Ẩn hoàn toàn brand Cloudy Learning, thay bằng logo & màu sắc riêng |
| Tenant Isolation | Dữ liệu, người dùng, khoá học hoàn toàn tách biệt giữa các tenant |
| Tenant Admin Panel | Bảng quản trị riêng cho từng tổ chức |

---

## 🔌 API & Tích hợp

| Tính năng | Mô tả |
|-----------|-------|
| REST API Access | API đầy đủ cho courses, users, progress, quizzes… |
| API Key Management | Tạo & thu hồi API key cho từng đối tác / ứng dụng bên ngoài |
| Webhook | Gửi sự kiện thời gian thực (enrollment, completion, payment…) đến hệ thống ngoài |
| SSO / SAML Integration | Đăng nhập một lần qua hệ thống nội bộ (LDAP, Azure AD, Okta…) |
| LMS Integration | Kết nối chuẩn SCORM / xAPI với các LMS khác |

## 💰 Monetization nâng cao

| Tính năng | Mô tả |
|-----------|-------|
| Affiliate / Referral System | Chia hoa hồng tự động khi giới thiệu người dùng / khoá học mới |
| Dynamic Pricing | Giá thay đổi theo vùng địa lý, nhóm người dùng, hoặc thời gian |
| Bundle / Package | Đóng gói nhiều khoá học thành combo với giá ưu đãi |
| B2B Licensing | Bán licence cho doanh nghiệp mua khoá học cho nhóm nhân viên |
| Invoice & Tax Management | Xuất hoá đơn, tính thuế VAT theo quốc gia |
